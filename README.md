**A magyar szimulátoros motorsport új korszaka**

## 🚦 Projekt leírása

A **HUN-SRT Racing Manager** egy webes alkalmazás, amelyet a **HUN-SRT (Hungarian Sim Racing Team)** e-sport versenyliga számára fejlesztettünk. A platform célja, hogy központi hubként szolgáljon a versenyzők, adminisztrátorok és rajongók számára egyaránt.

📆 **2024-ben alakult liga**  
🌍 **Valódi együttműködés** e-sport csapattal  
🧠 **Modern technológiák** használata (Firebase, Cloudinary, React)  
🔥 **2025-ben már több száz pilóta és futam**  

---

## 🎮 Funkciók

| Funkció | Leírás |
|--------|--------|
| 🏠 **Főoldal** | Liga bemutatása, történet, küldetés, közösségi média |
| 📅 **Futamok** | Közelgő futamok listázása, szűrés, adminisztráció |
| 🏆 **Bajnokságok** | Összes elérhető kategória megtekintése (ACC, AC, F1 23/24 stb.) |
| 💬 **Fórum** | Közösségi beszélgetések, tippek, stratégiák, aktuállis frissitések|
| 👤 **Profil** | Felhasználói adatok módosítása |
| 🛡️ **Szabályzat** | Versenyszabályok, viselkedési kódex, Discord név egyezés stb. |
| 🔐 **Admin ** | Futamok kezelése, felhasználók kezelése, fórum moderálás |

## 🧪 Használt technológiák

| Technológia | Leírás |
|-------------|--------|
| 🟦 **React.js** | Komponensalapú, gyors frontend fejlesztés |
| 🔥 **Firebase** | Autentikáció, adatbázis, hosting |
| ☁️ **Cloudinary** | Képfeltöltés és optimalizálás |
| 🎨 **Tailwind CSS** | Gyors és testreszabható stílus |
| 📦 **React Query** | Hatékony adatkezelés |
| 🛠️ **Vite** | Szupergyors fejlesztői környezet |
| 🌐 **Responsive Design** | Mobilra és asztalira optimalizálva |
